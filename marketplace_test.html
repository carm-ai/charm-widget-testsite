Marketplace widget embed test

This Widget example shows a quick POC implementatin of a full Javascript Marketplace Partner Widget built on the CHARM Platform widget framework developed for Abrigo. All rendering is done in Angular, with the code being deployed as one standalone javascript file loaded into the end-users browser. The implementation is not complete, tested or optimized, but shows the concept of using the CHARM Widget framework to develop a free-standing javascript widget that can be embedded into Marketplace partner sites.

Data is fetched directly from the CHARM Platform integration API using the new API:s available for Entity-Lender-Bid workflows. The three offers are in the example just statically entered into the CHARM Platform database, for a real implementation bids need to be fetched from a Marketplace matching engine.

Embedding for the partner will be very simple, using the custom my-smb-score HTML tag. The partner simply need to embed this tag and reference an entity to show a Score and Offfers for. for example;

<my-smb-score init="test_tenant_1" entity-id="90246279">
